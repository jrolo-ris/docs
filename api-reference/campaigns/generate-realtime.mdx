## Description

Initiates the generation of a video for a specific campaign with realtime playback. This endpoint synchronously triggers a video render and returns a streamId.

<Note>

- The `fields` array must match the expected fields for the campaign's template.
- The `callback_url` will be called when the video is ready to begin streaming.
</Note>
